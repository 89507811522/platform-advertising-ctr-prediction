# Модель предсказания CTR (кликов по рекламе)

Проект машинного обучения для предсказания вероятности кликов на рекламные объявления.

## Основные файлы
- `PlatformAdvandex.ipynb` - основной ноутбук с полным пайплайном
- `requirements.txt` - зависимости Python
- Модели (.joblib) и данные (.csv) исключены из Git

## Метрики модели
- PR-AUC: 0.3742 (до калибровки), 0.3607 (после)
- Accuracy: 82.89%
- Brier Score: 0.1287

## Технологии
Python, Scikit-learn, Pandas, Jupyter
